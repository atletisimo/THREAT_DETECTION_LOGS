# THREAT_DETECTION_LOGS

Овој проект  анализира и споредува класични алгоритми за машинско учење за детекција на сајбер закани врз основа на синтетичкиот датасет. 

Вклучува протоколи како:

#### TCP, UDP, ICMP, HTTP и HTTPS,

#### заедно со ознаки за акција (allowed/blocked), 

#### класификации на закани (benign, suspicious, malicious),

#### и метаподатоци како изворни/дестинациски IP адреси, user agents и патеки на барања (request paths). 

Проектот дополнително ќе примени semi-supervised граф-базирани алгоритми (Label Propagation) и Graph Neural Networks (GNN) за да се детектираат потенцијални закани и аномалии во мрежата.

Овој датасет може да се користи за тренирање на модели на машинско учење за системи за:
#### детекција на упади (IDS), 
#### детекција на аномалии и
#### класификација на закани.
